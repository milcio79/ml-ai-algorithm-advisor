# Code of Conduct

This project aims to be useful, respectful, and practical.

Participants are expected to:

- Be respectful and constructive.
- Assume good intent.
- Avoid harassment, insults, or discriminatory language.
- Keep feedback focused on the work.
- Help maintain a welcoming environment for users with different levels of ML and software experience.

Maintainers may remove comments, issues, or pull requests that violate these expectations.
